## 📋 Sobre o Projeto

Este é um projeto educacional desenvolvido para ensinar os fundamentos do GitHub de forma prática e visual. O site apresenta uma interface moderna com design gradiente roxo, animações suaves e uma estrutura didática que facilita o aprendizado.

### ✨ Características Principais

- 🎨 **Design Moderno**: Interface com gradientes vibrantes e animações suaves
- 📱 **Totalmente Responsivo**: Funciona perfeitamente em dispositivos móveis e desktop
- 🎯 **Conteúdo Didático**: Tópicos organizados do básico ao avançado
- 💫 **Efeitos Interativos**: Cards com hover effects e transições elegantes
- 🌐 **SEO Otimizado**: Estrutura HTML semântica e meta tags apropriadas
- ⚡ **Performance**: Carregamento rápido com CSS puro (sem frameworks pesados)

## 🚀 Como Executar

1. **Clone o repositório**
   ```bash
   git clone https://github.com/seu-usuario/aprenda-github.git
   ```

2. **Navegue até a pasta do projeto**
   ```bash
   cd aprenda-github
   ```

3. **Abra o arquivo HTML**
   - Dê um duplo clique em `index.html`, ou
   - Use um servidor local como o Live Server (VS Code)
